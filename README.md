# Chrome 便携版智能升级器 (Chrome_Portable_Updater)

一个专为 Windows 用户（特别是国内无代理网络环境）设计的、免翻墙的绿色便携版 Google Chrome 浏览器全自动升级与一键引导启动器。

本工具已全面升级，彻底废弃了以往的 Bat 与 PowerShell 脚本方案，采用**原生 C# WinForms** 重新开发，旨在提供更低的系统占用、更高的运行稳定性以及 100% 绿色无污染的沙盒隔离体验。

---

## 🌟 核心特性

- **🚀 100% 绿色便携隔离**
  支持彻底的沙盒隔离。引导器启动时会自动注入 `--user-data-dir` 相对路径参数，将所有浏览器缓存、用户数据、扩展插件和配置文件牢牢锁定在同级目录下的 `UserData` 文件夹中。绝对不污染 C 盘，即插即用。
  
- **🔒 国内网络多路容灾（免翻墙直连）**
  针对国内复杂的网络封锁环境，开发了三路自动顺延探测下载引擎：
  1. 优先尝试直连 **Google Omaha 官方更新接口**；
  2. 若官方接口超时受阻，自动无缝切换至**阿里巴巴国内高速 NPM 镜像源**解析最新版本号；
  3. 若镜像源受限，自动切换至**异次元 (iPlaySoft) 网页解析源**提取最新版本。
  解析成功后，直接向谷歌国内 CDN 下载服务器（`dl.google.com`）发起完整离线包下载，免翻墙，速度拉满。

- **📐 高精度版本比对算法**
  废弃了传统的“绝对相等”校验，引入了高精度的数学版本比对算法（支持 4 段式版本号）。若本地版本大于或等于服务器最新版本，则智能判定为最新版并直接拦截更新，彻底终结了以往脚本方案中常见的“无限循环下载”隐患。

- **📦 智能“俄罗斯套娃”解压技术**
  完美兼容谷歌最新的通用安装程序（GoogleUpdater）高强度嵌套压缩包结构。程序通过递归算法自动“剥洋葱”般静默穿透 4 层嵌套目录，自动定位、解压并部署最深层的 `Chrome-bin` 运行文件，完美向下兼容历史版本。

- **🎨 现代、精致、安全的 UI 设计**
  - **自动系统检测**：启动时自动读取注册表底层硬件属性，精准锁定 64位/32位 系统并追加自动检测提示，解决 Windows 系统版本伪报问题。
  - **模态弹出进度层**：下载与静默解压过程中，采用独立的模态遮罩进度窗口（禁止手动关闭），并带有带感的呼吸闪烁提示。在更新成功后，会发出悦耳的系统提示音并自动关闭，安全可靠。
  - **扁平化设计**：界面采用微软雅黑字体、扁平化彩色高饱和度按钮以及高防边缘圆角剪裁。

## 🛠️ 如何开始使用

1. 下载最新发布的 `ChromeUpdater.exe` 并将其放入一个空白文件夹（例如 `D:\Chrome_Portable\`）。
2. 双击运行 `ChromeUpdater.exe`，根据需要调整以下参数（默认已自动匹配最推荐的参数）：
   - 系统架构（32位 / 64位 自适应）
   - 版本类型（稳定版 / 测试版 / 开发版 / 金丝雀版）
   - 是否关联为默认浏览器
   - 是否创建桌面快捷方式
3. 点击 **【检查并更新】**，程序将全自动部署完毕。
4. 部署完成后，以后只需双击运行该工具或生成的桌面快捷方式，即可直接启动绿色隔离版 Chrome。

- **依赖说明**：本工具不依赖任何第三方动态链接库（DLL）或第三方 NuGet 依赖包，完全采用 .NET 原生网络和多线程库构建，编译后即为纯粹的单个 EXE 绿色文件。
