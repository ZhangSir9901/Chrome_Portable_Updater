## ✨ 核心特性：四大黄金定律

### 🛡️ 1. 网络层的“三路容灾”与“直连国内节点”
*   **免翻墙极速下载**：彻底摒弃国外容易被阻断的源。版本解析直连阿里巴巴国内高速镜像源（npmmirror.com），安装包下载直达谷歌国内 CDN 节点（dl.google.com），下载速度瞬间拉满。
*   **防 403 与反拦截**：底层物理屏蔽本地代理（Proxy = null）并强制注入合法 User-Agent，无视本地 Clash/V2ray 代理干扰，彻底防止网络请求被挂死。

### 🎨 2. 界面层的“无感切换”与“GPU 硬件加速”
*   **极美模态遮罩层**：专为同窗口开发设计的 Overlay 模态遮罩层，替代传统丑陋、繁琐的多窗口弹窗，优雅锁死主进程操作。
*   **60帧极致动画**：进度条彩蛋不使用定时器重绘，直接调动 WPF 原生的 Storyboard 引擎与 GPU 3D 渲染，文字滚动自带发光特效，纵享 60 帧极度丝滑。

### ⚙️ 3. 系统层的“双规注册表”与“全系统接管”
*   **跨越 Win7 到 Win11 的默认浏览器接管**：
    *   *Win10/Win11*：静默写入专属 ProgID（"Google Chrome 便携版"）上系统户口，智能呼出系统设置面板并执行 30秒 无感异步轮询。
    *   *Win7/Win8*：自动降级为暴力注入模式，瞬间强改底层关联，无需用户干预。
*   **归一化路径匹配**：底层 IO 与注册表查验严格实行绝对路径的小写正斜杠归一化，判定成功率 100%。

### ⚡ 4. 架构层的“异步非阻塞”与“零 DLL 依赖单文件”
*   **UI 线程绝对霸权**：凡是涉及网络请求、大文件 IO、解压、注册表写入等耗时操作，全部强制挂入 `await Task.Run()` 后台开辟。主线程只管渲染，软件永远不会出现“无响应”。
*   **100% 独立单文件设计**：物理去除了对外部 `System.Text.Json` 的引用，改用自研的纯原生正则表达式（Regex）JSON 参数注入引擎。项目编译后**仅生成一个独立的 `.exe` 程序**，无需附带任何 DLL 即可单文件完美运转，实现极致便携。
*   **一处控制，三方绝对同步**：在代码设计上全线采用 C# 反射机制，完全去除了硬编码的版本控制常量。升级版本时，你无需更改任何主代码，只需在 `Properties/AssemblyInfo.cs` 中修改一行日期文本（如 `"2026-07-14"`），编译后**系统文件详细属性、前台版权 UI、后台判定逻辑**三方将绝对自动同步！

## 🤖 深度隐私：AI 瘦身体检与内核级彻底阻断

为应对 Chrome 频繁强制更新带来的本地 AI（Gemini / Glic）静默下载大垃圾文件、强推 AI 工具栏等痛点，程序深度集成了 AI 自动侦测与净化防御模块：

### 1. 「AI 模型体检」极客清理功能
程序会自动对**便携版私有目录**与**系统 C 盘原生目录**进行双向检索，扫描 `OptGuideOnDeviceModel` 缓存：
*   **未检测到浏览器**：按钮显示为 **「AI 模型体检」**（按钮中灰色，自动安全置灰，不可点击）。
*   **纯净无 AI 垃圾**：按钮自动呈现优雅的**皇家紫 (`#9B59B6`)**（状态启用，点击无多余反应，让强迫症用户 100% 放心）。
*   **存在 AI 大垃圾**：按钮自动转为醒目的**深橙色 (`#D35400`)**，鼠标悬停显示悬浮提示：`“点击可清理！！！”`。
    *   *点击后，程序将毫秒级计算缓存大小，提示是否物理粉碎（自动换算为 **MB/GB** 级别直观展现），并同步强力注入拦截参数，彻底锁死二次静默下载的可能性。*

### 2. 内核级封锁 Chrome 146+ 全域 AI 接口
不仅阻断后台偷偷下载大文件，更从内核层、UI 界面层将 AI 暗门剥离干净。自动在 `Local State` 中注入封锁参数：
*   **`"glic@2"`**：彻底禁用 Chrome 146+ 新版强推的标签栏/工具栏右侧的 **“问问 Gemini (glic)”** 浮动呼出按钮与所有右键相关 AI 菜单。
*   **`"prompt-api-for-gemini-nano@2"`** 与 **`"summarization-api-for-gemini-nano@2"`**：彻底关闭网页端 Prompt API 及 Summarization 接口，防止第三方恶意网页后台强制唤起大模型下载。
*   **`"ai-mode-omnibox-entrypoint@2"`**：隐藏地址栏搜索框右侧多余的 Gemini 快捷入口。
*   **`"compose@2"`**：全局关闭 Help me write 辅助撰写组件。

### 3. 智能「Chrome 进程锁」安全探测
由于 Chrome 在运行期间会完全锁定配置文件、并在退出时反写覆盖，升级器在注入调优参数前会自动检索系统中的 `chrome` 进程。如果检测到浏览器正在后台运行，会主动弹出友好提示，引导用户关闭 Chrome 后再继续， 100% 保证屏蔽策略的安全写入与不被回写覆盖。
